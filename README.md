# Sistema de Locadora de Filmes

## Descrição

Este projeto consiste em um sistema de gerenciamento de locadora de filmes desenvolvido em Java.
A aplicação permite cadastrar, listar, alugar, devolver e remover filmes, utilizando tanto interface de console quanto interface gráfica (Swing).

O objetivo do projeto é aplicar conceitos fundamentais de programação orientada a objetos (POO) e boas práticas de desenvolvimento.

---

## Funcionalidades

* Cadastro de filmes
* Listagem de filmes
* Busca por ID
* Aluguel de filmes
* Devolução de filmes
* Remoção de filmes
* Interface gráfica com Java Swing
* Modo console interativo

---

## Tecnologias Utilizadas

* Java (JDK 17 ou superior)
* Java Swing (interface gráfica)
* VS Code
